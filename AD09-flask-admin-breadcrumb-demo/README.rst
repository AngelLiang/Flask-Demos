给 flask-admin 添加 breadcrumb
################################


准备工作
***********

首先需要安装npm和bower::

    $ npm install -g bower

编译::

    $ cd app/static/sb-admin-2
    $ bower install


快速开始
*********

::

    pipenv install
    pipenv run flask run
